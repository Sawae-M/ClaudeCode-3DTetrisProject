====================================================
  立体テトリス (TetrisCube3D) 仕様書
====================================================

■ 基本ルール
  - 5×5×5 のキューブにブロックを詰める
  - 5×5 の一面が揃うと消える
  - 消えた面の法線方向に 1 マス分ブロックが落ちる（シフトダウン）
  - 複数面同時消去の優先順位：上 → 下 → 左 → 右 → 前 → 後
  - ブロックがキューブ範囲外に出たらゲームオーバー

■ ピース一覧（7種類、回転なし）
  ① I型        直線4連 (1×1×4)
  ② 十字型     XZ平面 十字5マス
  ③ S型        水平 S字
  ④ L型        L字3連
  ⑤ S型反転   ③の鏡像 (Z型)
  ⑥ 3Dジグザグ XY方向に段差
  ⑦ 3Dジグザグ反転  ⑥の鏡像

■ 操作
  - ピースは移動のみ（回転なし）
  - キューブ全体を 90° 回転 → 落下方向（重力方向）が変わる
  - ブロック落下中もキューブ回転可能
  - ボタン長押しで落下加速

■ キューブ回転と重力方向
  重力方向 6 種：Down / Up / Left / Right / Forward / Back
  キューブ回転で重力方向が切り替わる。
  ピースは常に「現在の重力方向」に沿って落下する。

■ 視点
  - 6 方向固定カメラを切り替えて使用
    ( 上 / 下 / 右 / 左 / 手前 / 奥 )
  - いずれもキューブ中心を注視

■ 演出・UI
  - 落下地点を半透明のゴーストブロックで表示
  - 邪魔ブロック：一定時間ごとにランダム位置へ落下
      → 1 ターン前に半透明の警告表示
  - スコア、レベル、NEXT ピースを HUD に表示

■ 難易度
  - 時間経過で段階的に落下速度上昇
  - 初速 1.5 秒/マス → 最高速 0.3 秒/マス（上限あり）

■ ゲームオーバー条件
  - 新規ピースのスポーン位置がすでに埋まっている
  - または面消去後のシフトでブロックがキューブ外に出る

====================================================
  実装フェーズ
====================================================

Phase 1 : 5×5×5 グリッド表示
Phase 2 : キューブの回転と視点切り替え
Phase 3 : ピースの落下と移動
Phase 4 : 面消去とブロック落下（シフト）
Phase 5 : ピース 7 種類の定義
Phase 6 : 邪魔ブロック
Phase 7 : 難易度上昇
Phase 8 : UI・演出
